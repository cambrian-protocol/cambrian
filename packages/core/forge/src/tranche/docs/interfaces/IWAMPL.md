## `IWAMPL`

### `deposit(uint256 amples) → uint256` (external)

Transfers AMPLs from {msg.sender} and mints wAMPLs.

### `burn(uint256 wamples) → uint256` (external)

Burns wAMPLs from {msg.sender} and transfers AMPLs back.
